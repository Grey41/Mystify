## Is it safe?

Mystify does not collect passwords or personal information. All calculations are performed locally in your browser.

## Can I get banned for using it?

Mystify aims to comply with Mystera Legacy's Terms of Service. It does not automate gameplay or interact with the server. If you are unsure whether a feature complies with the game's rules, review the game's policies or contact the developers directly.
